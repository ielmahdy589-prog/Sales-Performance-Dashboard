# Sales Performance Dashboard

## Overview
This Power BI dashboard provides a detailed analysis of company sales performance. 
It was built from raw data using **Power Query for data cleaning and transformation**, and advanced **Data Modeling** techniques to split tables into fact and dimension tables for efficient analysis. 

The dashboard includes key metrics like Total Sales, Total Profit, Total Orders, Top Ship Day, and Top Products, along with detailed breakdowns by Region, Segment, Category, and Sub-Category.

## Features
- **Data Preparation:** Used Power Query to clean and transform raw sales data.
- **Data Modeling:** Split the main sales table into fact and dimension tables for optimized performance.
- **Interactive Metrics:** Total Sales, Total Profit, Total Orders, Top Ship Day, and Top Products using DAX.
- **Trends Analysis:** Monthly sales and profit trends visualized with line charts.
- **Categorical Analysis:** Pie charts and bar charts for Region, Segment, Category, and Sub-Category.
- **Dynamic Filters:** Year, Region, and Sub-Category slicers for interactive insights.

## Tools & Techniques
- **Power BI Desktop:** Main dashboard development.
- **Power Query:** Data cleaning, transformation, and table splitting.
- **DAX:** Calculations for Total Sales, Growth %, Top Products, and Top Ship Day.
- **Data Modeling:** Fact & dimension tables, relationships, and optimized queries.
- **Excel/CSV:** Original data source.

## Insights
- Quickly identify top performing products and regions.
- Track sales growth month over month using calculated DAX measures.
- Analyze high-performing shipping days to optimize logistics.
- Monitor category and sub-category performance for strategic decision making.
